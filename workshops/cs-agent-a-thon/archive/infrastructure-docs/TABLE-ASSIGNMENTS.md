## Table Tent Card Preparation Notes

### For Each Table:
- Print **2 team tent cards** (one for each team of 5)
- Include:
  - Table Number
  - Team Number
  - Team Type (Advanced/Mixed)
  - List of 5 fictitious names

### Card Format Recommendation:
```
┌─────────────────────────────────┐
│  TABLE 1 - TEAM 1               │
│  (Advanced)                     │
│                                 │
│  • Rayna Bacheva                │
│  • Chanok Rattanakoson          │
│  • Jatindra Sanyal              │
│  • Hiwa Saywan                  │
│  • Olha Davydenko               │
└─────────────────────────────────┘
```

### Printing Instructions:
- **Total Cards Needed**: 24 team cards (2 per table × 12 tables)
- **Card Size**: Standard table tent (8.5" × 11" folded)
- **Font**: Large, readable (18-24pt for names)
- **Color Coding**: Consider different colors for Advanced (Tables 1-2) vs Mixed (Tables 3-12)
